# metodos-numericos-python
étodos Numéricos en Python: Integración y Raíces de Ecuaciones Implementación de algoritmos fundamentales de cálculo numérico, incluyendo Integración de Riemann, Método de Bisección y Newton-Raphson. El proyecto se enfoca en la evaluación dinámica de funciones y el análisis del error relativo frente a soluciones analíticas.
